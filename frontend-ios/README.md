# frontend-ios/

SwiftUI iOS client for Offside CRM. The Xcode project is **generated by xcodegen** — the source-of-truth is `Project.yml`. `OffsideCRM.xcodeproj/` is gitignored; every contributor regenerates it locally.

## Prerequisites

- Xcode 16+ on macOS
- iOS 17+ SDK
- xcodegen — `brew install xcodegen`

## Generate the Xcode project

From the repo root:

```bash
pnpm ios:gen        # runs xcodegen inside frontend-ios/
open frontend-ios/OffsideCRM.xcodeproj
```

Or directly:

```bash
cd frontend-ios
xcodegen generate
open OffsideCRM.xcodeproj
```

## Layout

```
frontend-ios/
├─ Project.yml              # xcodegen source-of-truth — edit this, not the .xcodeproj
├─ OffsideCRM/              # Swift sources
│  ├─ OffsideCRMApp.swift   # @main entry
│  └─ ContentView.swift     # initial view (M0 placeholder, brand parity validated)
├─ Generated/               # OpenAPI Swift client — generated by tools/openapi/sync-ios.sh (gitignored)
└─ README.md
```

## Real iOS scope lands in M6

This M0 scaffold only proves the project builds. **M6** (iOS shell + auth + read-only views) lands the real surface:

- Login screen + JWT in Keychain
- OpenAPI-generated Swift client (auto-synced into `Generated/`)
- List + detail screens for Contacts / Companies / Deals / Tasks
- Offline cache for last 50 visited records
- APNs registration (push payloads consumed in M13)
- Pull-to-refresh, skeletons, offline banner

**M13** then layers HITL approval push notifications with lock-screen Approve / Reject / Snooze actions.
